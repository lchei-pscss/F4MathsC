1.1 基本形式 (General Form)

所有二次方程都可以寫成以下的一般式：

$$ax^2 + bx + c = 0$$

其中 $a \neq 0$。解方程的目的就是找出能令等式成立的未知數 $x$ 的值。

1.2 方法一：因式分解法 (Factorization)

適用時機： 當常數項與 $x$ 的係數可以輕易透過「十字相乘法」分解時。

核心原理： 若 $A \times B = 0$，則 $A = 0$ 或 $B = 0$。

📝 經典例題 (實根)： 解方程 $x^2 - 5x + 6 = 0$

步驟 1： 十字相乘因式分解

$$(x - 2)(x - 3) = 0$$

步驟 2： 令各個括號為 $0$

$x - 2 = 0 \quad \text{或} \quad x - 3 = 0$

答案： $x = 2 \quad \text{或} \quad x = 3$

1.3 方法二：求根公式 (Quadratic Formula)

適用時機： 任何二次方程！特別是無法輕易因式分解（答案有根號或小數）的時候。

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

其中 $\Delta = b^2 - 4ac$ 稱為判別式 (Discriminant)：

  • $\Delta > 0$：兩個不相存的實根
  • $\Delta = 0$：一個二重實根（兩個相等的實根）
  • $\Delta < 0$：沒有實根（在實數範圍內）
📝 經典例題 (實根)： 解方程 $2x^2 - 5x + 3 = 0$

步驟 1： 找出 $a, b, c$ 的值

$a = 2, \ b = -5, \ c = 3$

步驟 2： 代入求根公式

$$x = \frac{-(-5) \pm \sqrt{(-5)^2 - 4(2)(3)}}{2(2)}$$

$$x = \frac{5 \pm \sqrt{25 - 24}}{4}$$

$$x = \frac{5 \pm \sqrt{1}}{4}$$

步驟 3： 分開計算兩個根

$$x = \frac{5 + 1}{4} = 1.5 \quad \text{或} \quad x = \frac{5 - 1}{4} = 1$$

答案： $x = 1.5 \quad \text{或} \quad x = 1$

1.4 進階：當 $\Delta < 0$ 時的複數根

當我們代入求根公式遇到 $\Delta < 0$ 時，可以利用虛數單位轉換：$\sqrt{-k} = \sqrt{k} \cdot \sqrt{-1} = \sqrt{k}i$ 來求解。

📝 經典例題 (虛根)： 解方程 $x^2 - 4x + 13 = 0$

步驟 1： 計算判別式 $\Delta$

$$\Delta = (-4)^2 - 4(1)(13) = 16 - 52 = -36$$

因為 $\Delta = -36 < 0$，方程擁有兩個複數根。

步驟 2： 代入求根公式

$$x = \frac{-(-4) \pm \sqrt{-36}}{2(1)}$$

利用虛數單位轉換：$\sqrt{-36} = 6i$

$$x = \frac{4 \pm 6i}{2}$$

步驟 3： 約分化簡

$$x = 2 \pm 3i$$

答案： $x = 2 + 3i \quad \text{或} \quad x = 2 - 3i$

⚠️ 重要觀察： 二次方程的兩個虛根必定互為共軛複數！

1.5 根與係數的關係 (Relation between Roots and Coefficients)

設 $\alpha$ 及 $\beta$ 為二次方程 $ax^2 + bx + c = 0$ 的兩個根。我們不需真正解出 $\alpha$ 和 $\beta$，就能直接從方程的係數求得它們的「和」與「積」：

$$\text{二根之和 (Sum of Roots)}: \quad \alpha + \beta = -\frac{b}{a}$$ $$\text{二根之積 (Product of Roots)}: \quad \alpha\beta = \frac{c}{a}$$

💡 DSE 必考熱門對稱代數式變形

考試經常要求利用二根之和及二根之積來計算其他代數式，以下兩個公式必須熟背：

  • 平方和： $\alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta$
  • 倒數和： $\frac{1}{\alpha} + \frac{1}{\beta} = \frac{\alpha + \beta}{\alpha\beta}$
📝 經典例題： 設 $\alpha$ 及 $\beta$ 為方程 $x^2 - 4x + 2 = 0$ 的兩根。求下列各式的值：
(a) $\alpha + \beta$ 及 $\alpha\beta$
(b) $\alpha^2 + \beta^2$
(c) $\frac{1}{\alpha} + \frac{1}{\beta}$

解題步驟：

由方程可知：$a = 1, \ b = -4, \ c = 2$。

(a) 代入基本公式：

$$\alpha + \beta = -\frac{-4}{1} = 4$$

$$\alpha\beta = \frac{2}{1} = 2$$

(b) 利用平方和公式：

$$\alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta$$

$$\alpha^2 + \beta^2 = (4)^2 - 2(2) = 16 - 4 = 12$$

(c) 通分合併倒數和：

$$\frac{1}{\alpha} + \frac{1}{\beta} = \frac{\alpha + \beta}{\alpha\beta} = \frac{4}{2} = 2$$

2.1 數系的擴展 (Extension of Number System)

在實數範圍內，負數不能開平方。為了解決這個系統性的限制，數學家引入了虛數單位 (Imaginary Unit) $i$：

$$i = \sqrt{-1} \quad \implies \quad i^2 = -1$$

有了 $i$ 之後，數系就從實數 ($\mathbb{R}$) 擴展到了複數 ($\mathbb{C}$)，使所有多項式方程都有解。

2.2 虛數單位 $i$ 的次方規律

$i$ 的正整數次方具有非常奇妙的每 4 次一循環的週期性特性：

次方 計算過程 最簡結果
$i^1$ $i$ $i$
$i^2$ $\sqrt{-1} \times \sqrt{-1}$ $-1$
$i^3$ $i^2 \times i = -1 \times i$ $-i$
$i^4$ $i^2 \times i^2 = (-1) \times (-1)$ $1$

💡 速算技巧： 將指數除以 $4$ 看餘數。若餘數為 $r$，則 $i^n = i^r$。

📝 範例： 求 $i^{2026}$ 的值。

因為 $2026 \div 4 = 506 \dots \text{餘 } 2$

所以 $$i^{2026} = i^2 = -1$$

2.3 複數的概念與共軛複數 (Conjugate)

一個完整的複數 $z$ 可以寫成標準式：

$$z = a + bi$$

其中 $a$ 稱為實部 (Real Part)，$b$ 稱為虛部 (Imaginary Part)，且 $a, b \in \mathbb{R}$。

☯️ 共軛複數 (Complex Conjugate)

若有一個複數 $z = a + bi$，它的共軛複數記作 $\bar{z}$，定義為虛部變號：

$$\bar{z} = a - bi$$

🔍 共軛複數的重要運算性質

  • 相加為實數： $z + \bar{z} = (a + bi) + (a - bi) = 2a$
  • 相乘為實數： $z \cdot \bar{z} = (a + bi)(a - bi) = a^2 - (bi)^2 = a^2 + b^2$ （平方和公式）

2.4 複數的基本四則運算 (Operations of Complex Numbers)

複數的運算核心法則：將 $i$ 視為普通未知數進行同類項合併，但切記只要遇到 $i^2$ 就要代換為 $-1$。

1. 複數相加 / 相減

法則： 實部與實部相加減，虛部與虛部相加減。

$$(a + bi) \pm (c + di) = (a \pm c) + (b \pm d)i$$
📝 例題 (相加)： 計算 $(3 + 4i) + (1 - 2i)$

$$= (3 + 1) + (4 - 2)i$$

$$= 4 + 2i$$

📝 例題 (相減)： 計算 $(3 + 4i) - (1 - 2i)$

$$= 3 + 4i - 1 + 2i \quad \text{(注意負負得正)}$$

$$= (3 - 1) + (4 + 2)i$$

$$= 2 + 6i$$

2. 複數相乘

法則： 使用彩虹乘法（分配律）展開，並將 $i^2$ 轉換為 $-1$ 後再次合併同類項。

$$(a + bi)(c + di) = ac + adi + bci + bdi^2 = (ac - bd) + (ad + bc)i$$
📝 例題： 計算 $(2 + 3i)(1 - 4i)$

$$= 2(1) + 2(-4i) + 3i(1) + 3i(-4i)$$

$$= 2 - 8i + 3i - 12i^2$$

$$= 2 - 5i - 12(-1) \quad \text{(因為 } i^2 = -1\text{)}$$

$$= 2 - 5i + 12$$

$$= 14 - 5i$$

3. 複數的分母有理化 (Division & Rationalization)

法則： 當分母含有虛數時，分子與分母同時乘以分母的共軛複數，利用平方差公式 $(c+di)(c-di) = c^2 + d^2$ 來消除分母的 $i$。

📝 例題： 將 $\frac{5 + 2i}{3 - i}$ 化簡為 $a + bi$ 的形式

步驟 1： 分子分母同乘以分母的共軛複數 $(3 + i)$

$$\frac{5 + 2i}{3 - i} = \frac{(5 + 2i)(3 + i)}{(3 - i)(3 + i)}$$

步驟 2： 分別展開分子與分母

$$\text{分子：} (5 + 2i)(3 + i) = 15 + 5i + 6i + 2i^2 = 15 + 11i - 2 = 13 + 11i$$

$$\text{分母：} (3 - i)(3 + i) = 3^2 - i^2 = 9 - (-1) = 10$$

步驟 3： 組合成標準複數式

$$\frac{13 + 11i}{10} = \frac{13}{10} + \frac{11}{10}i$$

3.1 直線的斜率 (Slope of a Straight Line)

斜率用符號 $m$ 表示，用來衡量一條直線的傾斜程度與方向。若已知直線上兩點 $P(x_1, y_1)$ 和 $Q(x_2, y_2)$，斜率公式為：

$$m = \frac{y_2 - y_1}{x_2 - x_1}$$

💡 觀念小提示：

  • $m > 0$：直線向右上方傾斜（遞增）。
  • $m < 0$：直線向右下方傾斜（遞減）。
  • $m = 0$：水平線。
  • 斜率不存在：垂直線（因為分母 $x_2 - x_1 = 0$）。

3.2 直線方程的三種經典形式

1. 點斜式 (Point-slope Form)

適用時機： 已知直線上的一點 $(x_1, y_1)$ 及其斜率 $m$。

$$y - y_1 = m(x - x_1)$$

2. 斜截式 (Slope-intercept Form)

適用時機： 已知直線的斜率 $m$ 及其 $y$ 截距 $c$（即直線交於 $(0, c)$）。

$$y = mx + c$$

3. 一般式 (General Form)

任何直線都可以整理成以下的形式，其中 $A, B$ 不同時為 $0$：

$$Ax + By + C = 0$$

💡 從一般式快速找關鍵數據：

  • 斜率 $$m = -\frac{A}{B}$$
  • $y$ 截距 $$c = -\frac{C}{B}$$
  • $x$ 截距 $$= -\frac{C}{A}$$

3.3 截距 (Intercepts)

截距是直線與坐標軸交點的位置（注意：截距是一個數值，而不是坐標）。

  • $x$ 截距： 令 $y = 0$，求出對應的 $x$ 值。
  • $y$ 截距： 令 $x = 0$，求出對應的 $y$ 值。
📝 例題： 求直線 $3x - 4y + 12 = 0$ 的 $x$ 截距與 $y$ 截距。

求 $x$ 截距： 令 $y = 0$

$$3x - 4(0) + 12 = 0 \implies 3x = -12 \implies x = -4$$

求 $y$ 截距： 令 $x = 0$

$$3(0) - 4y + 12 = 0 \implies -4y = -12 \implies y = 3$$

答案： $x$ 截距為 $-4$，$y$ 截距為 $3$。

3.4 平行線與垂直線 (Parallel & Perpendicular Lines)

設有兩條直線 $L_1$ 與 $L_2$，其斜率分別為 $m_1$ 與 $m_2$：

平行關係 ($\parallel$)

若兩直線平行，則它們的傾斜程度相同：

$$m_1 = m_2$$

垂直關係 ($\perp$)

若兩直線垂直，則它們的斜率相乘等於 $-1$：

$$m_1 \times m_2 = -1 \quad \implies \quad m_2 = -\frac{1}{m_1}$$
📝 經典考試題型： 求通過點 $A(2, 5)$ 且與直線 $L: 2x - 3y + 6 = 0$ 垂直的直線 $L'$ 的方程。

步驟 1：求已知直線 $L$ 的斜率 $m_1$

將 $L$ 化為斜截式：$3y = 2x + 6 \implies y = \frac{2}{3}x + 2$

所以，直線 $L$ 的斜率 $m_1 = \frac{2}{3}$。

步驟 2：利用垂直關係求 $L'$ 的斜率 $m_2$

因為 $L' \perp L$，所以 $$m_2 = -\frac{1}{m_1} = -\frac{1}{\frac{2}{3}} = -\frac{3}{2}$$

步驟 3：利用「點斜式」建立 $L'$ 的方程

已知點 $(2, 5)$，斜率 $m = -\frac{3}{2}$：

$$y - 5 = -\frac{3}{2}(x - 2)$$

兩邊同乘以 $2$：

$$2(y - 5) = -3(x - 2)$$

$$2y - 10 = -3x + 6$$

步驟 4：整理成一般式

$$3x + 2y - 16 = 0$$

答案： 直線方程為 $3x + 2y - 16 = 0$。

4.1 函數的記法與基本概念 (Notation of Functions)

函數（Function）就像一部自動販賣機：你輸入一個數值 $x$（自變量），它就會根據特定的規則，輸出唯一一個對應的答案 $f(x)$（應變量）。

例如，若規定規則為 $f(x) = 2x + 3$，當你輸入 $x = 5$ 時，輸出就是 $f(5) = 2(5) + 3 = 13$。

4.2 經典題型一：普通代入法 (Substitution)

核心秘訣： 把函數原本規則裡的所有 $x$ 改寫成括號 `( )`，再將要代入的數字或表達式塞進括號內。特別注意負數代入時的平方變化！

📝 例題： 設 $f(x) = 2x^2 - 3x + 1$。求 $f(-2)$ 的值。

解題步驟：

將所有的 $x$ 用 $(-2)$ 代替：

$$f(-2) = 2(-2)^2 - 3(-2) + 1$$

⚠️ 注意：$(-2)^2 = 4$（負負得正）；$-3 \times (-2) = +6$

$$f(-2) = 2(4) + 6 + 1$$

$$f(-2) = 8 + 6 + 1 = 15$$

答案： $f(-2) = 15$

4.3 經典題型二：DSE 卷二必考變形題 — $f(\text{表達式}) \implies f(k)$

題目特徵： 題目給出的不是標準的 $f(x)$，而是像 $f(x+1) = ax^2 + bx + c$ 這樣的複雜式子，卻要求你求某個特定數字 $f(k)$ 的答案。

🎯 DSE 卷二真題模擬：
設 $f(x + 2) = x^2 - 3x + 4$。求 $f(1)$ 的值。

⚡ 解法一：DSE 秒殺計數機速解法（極力推薦法）

在選擇題中，我們不需要辛苦找出 $f(x)$ 的一般式，只要讓「輸入端」相等即可！

我們的目標是求 $f(\color{red}{1})$，而已知給的是 $f(\color{blue}{x + 2})$。

令它們內部的東西相等：

$$\color{blue}{x + 2} = \color{red}{1} \quad \implies \quad x = -1$$

這代表：只要把 $x = -1$ 代入右邊的算式，得到的答案就是 $f(1)$！

$$f(1) = (-1)^2 - 3(-1) + 4$$

$$f(1) = 1 + 3 + 4 = 8$$

答案： $f(1) = 8$ （只需兩步，15秒內完工！）

層次二：構造一般式法（卷一傳統題/長題目觀念）

如果題目很壞，要求你找出 $f(x)$ 的完整二次多項式，我們就要用換元構造法：

步驟 1： 令新的未知數 $u = x + 2$，從而反解出 $x = u - 2$。

步驟 2： 把原本式子裡右邊的所有 $x$ 全替換成 $(u - 2)$：

$$f(u) = (u - 2)^2 - 3(u - 2) + 4$$

步驟 3： 將右邊用橫式展開並合併同類項：

$$f(u) = (u^2 - 4u + 4) - (3u - 6) + 4$$

$$f(u) = u^2 - 4u + 4 - 3u + 6 + 4$$

$$f(u) = u^2 - 7u + 14$$

步驟 4： 把符號換回慣用的 $x$，得到通用函數：

$$f(x) = x^2 - 7x + 14$$

驗算一下求 $f(1)$：$f(1) = 1^2 - 7(1) + 14 = 1 - 7 + 14 = 8$。答案完全相同！

5.1 二次函數的兩種表現形式

二次函數的圖像是一條拋物線 (Parabola)。在考試中，它主要以兩種形式呈現：

1. 一般式 (General Form)

$$y = ax^2 + bx + c \quad (a \neq 0)$$
  • $a > 0$：拋物線開口向上 ∪；$a < 0$：拋物線開口向下 ∩。
  • $c$：拋物線的 **$y$ 截距**（即圖像交於 $(0, c)$）。
  • 對稱軸方程： 直線 $$x = -\frac{b}{2a}$$
  • 頂點坐標： $$\left(-\frac{b}{2a}, f\left(-\frac{b}{2a}\right)\right)$$

2. 頂點式 (Vertex Form)

$$y = a(x - h)^2 + k$$
  • 從頂點式可以**一眼看出**拋物線的最關鍵幾何特徵：
  • 頂點坐標 (Vertex)： 坐標為 $(h, k)$。
  • 對稱軸方程 (Axis of Symmetry)： 直線 $$x = h$$。

5.2 核心工具：配方法 (Completing the Square)

配方法是將二次函數從「一般式」手動轉換為「頂點式」的純代數技巧（完全不需要微積分/導數）。

💡 配方法三大核心步驟：

  1. 抽出 $x^2$ 的係數 $a$，讓括號內 $x^2$ 的前方保持乾淨（係數為 1）。
  2. 看括號內 $x$ 係數的 **「一半」**，在括號內同時 **加上並減去** 這個一半的平方。
  3. 將前三項組合成完全平方式 $(x \pm h)^2$，並將剩餘常數項移出括號外簡化。
📝 經典例題： 將二次函數 $y = 2x^2 - 12x + 11$ 化為頂點式，並寫出其頂點坐標。

步驟 1：提出前兩項的 $x^2$ 係數 $2$

$$y = 2(x^2 - 6x) + 11$$

步驟 2：配方（$x$ 的係數是 $-6$，其一半是 $-3$，所以要補上 $(\pm 3^2)$）

$$y = 2(x^2 - 6x + 3^2 - 3^2) + 11$$

$$y = 2(\underbrace{x^2 - 6x + 9}_{\text{完美完全平方}} - 9) + 11$$

步驟 3：化成完全平方，並把 $-9$ 乘以外面的 $2$ 搬出來

$$y = 2[(x - 3)^2 - 9] + 11$$

$$y = 2(x - 3)^2 - 18 + 11$$

$$y = 2(x - 3)^2 - 7$$

答案： 頂點式為 $y = 2(x - 3)^2 - 7$，頂點坐標為 $(3, -7)$。

5.3 求極值 (Maximum / Minimum Value) 的原理與公式

🎯 代數壓制原理（不用導數）

因為在實數範圍內，任何實數的平方都必定大於或等於零：$$\text{即 } (x - h)^2 \ge 0$$

  • 當 $a > 0$（開口向上 ∪）時：$a(x - h)^2 \ge 0$。因此 $a(x - h)^2 + k$ 的**最小值**就是 $k$（此時括號內必須等於零，即 $x = h$）。
  • 當 $a < 0$（開口向下 ∩）時：$a(x - h)^2 \le 0$。因此 $a(x - h)^2 + k$ 的**最大值**就是 $k$（此時同樣在 $x = h$ 發生）。

⚡ DSE 選擇題快攻：頂點公式法

如果你在做卷二選擇題，不想花時間配方，可以直接背誦由一般式推導出的頂點公式：

頂點的 $x$ 坐標（對稱軸）：$$h = -\frac{b}{2a}$$

💡 超級小秘訣： 算出 $x = -\frac{b}{2a}$ 後，直接把它**代回原方程**求 $y$ 值，這個 $y$ 值就是極值 $k$，完全不需要背複雜的 $\frac{4ac-b^2}{4a}$ 公式！

5.4 常見應用題型：幾何面積最值問題

DSE 卷一 Section A 常用這類題目來考核學生的建模型能力。通常是限制周長，要求最大面積。

📝 經典考題： 某農夫有一條長 $40 \text{ m}$ 的圍欄。他想用這條圍欄在空地上圍成一個矩形的菜園。求菜園可圍成的最大面積。

步驟 1：設未知數並建立函數模型

設矩形菜園的寬度為 $x \text{ m}$。

因為周長為 $40 \text{ m}$，所以「長 + 寬 = $20 \text{ m}$」，則菜園的長度可表示為 $(20 - x) \text{ m}$。

令菜園的面積為 $A \text{ m}^2$：

$$A = x(20 - x)$$

$$A = -x^2 + 20x$$

步驟 2：利用配方法或公式求最值

這裡使用**頂點公式速算法**（因為 $a = -1, B = 20$）：

當寬度 $$x = -\frac{b}{2a} = -\frac{20}{2(-1)} = 10 \text{ m}$$ 時，面積有最大值。

步驟 3：計算最大面積值

將 $x = 10$ 代入面積方程：

$$A_{\text{max}} = -(10)^2 + 20(10)$$

$$A_{\text{max}} = -100 + 200 = 100 \text{ m}^2$$

⚠️ （若用配方驗證：$A = -(x^2 - 20x) = -(x^2 - 20x + 10^2 - 10^2) = -(x - 10)^2 + 100$）

答案： 菜園可圍成的最大面積為 $100 \text{ m}^2$。

5.5 圖解二次方程與判別式 (Graphical Solution & Discriminant)

二次方程 $ax^2 + bx + c = 0$ 的實根，在幾何上正是二次函數 $y = ax^2 + bx + c$ 的圖像與 $x$ 軸的交點 ($x$-intercepts)。

① $\Delta > 0$ (2個實根)
② $\Delta = 0$ (1個二重根)
③ $\Delta < 0$ (無實根)

6.0 ASTC 像限 (ASTC Quadrants)

當角 $\theta$ 的大小跨越 $90^\circ$、$180^\circ$ 或 $270^\circ$ 時，三角函數值的正負號會隨其所在的**像限 (Quadrant)** 而改變。我們可以使用 **ASTC 法則**快速記憶：

像限 角度範圍 ($\theta$) 正值函數 (ASTC) 坐標正負 $(x, y)$
第一像限 (Q1) $0^\circ < \theta < 90^\circ$ All (全部均為正數) $(+, +)$
第二像限 (Q2) $90^\circ < \theta < 180^\circ$ Sin ($\sin \theta$ 為正) $(-, +)$
第三像限 (Q3) $180^\circ < \theta < 270^\circ$ Tan ($\tan \theta$ 為正) $(-, -)$
第四像限 (Q4) $270^\circ < \theta < 360^\circ$ Cos ($\cos \theta$ 為正) $(+, -)$

💡 口訣記憶法： 「All-Sugar-To-Coffee」或「常青樹（A-S-T-C）」，順時針或逆時針由第一至第四像限排列。

6.1 三角學定義 (Trigonometric Definitions)

在直角坐標平面上，設 $\theta$ 為一個標準位置角，其終邊上任意一點的坐標為 $P(x, y)$，且該點到原點的距離（半徑）為 $r = \sqrt{x^2 + y^2}$ （注意：$r$ 恆為正數）：

$$\sin \theta = \frac{y}{r} \quad , \quad \cos \theta = \frac{x}{r} \quad , \quad \tan \theta = \frac{y}{x}$$
📝 經典例題： 已知角 $\theta$ 的終邊通過坐標點 $P(-3, 4)$，求 $\sin \theta$ 及 $\tan \theta$ 的值。

步驟 1：計算點 $P$ 到原點的距離 $r$

$$r = \sqrt{(-3)^2 + 4^2} = \sqrt{9 + 16} = 5$$

步驟 2：根據定義代入坐標 $x = -3, \ y = 4, \ r = 5$

$$\sin \theta = \frac{y}{r} = \frac{4}{5}$$

$$\tan \theta = \frac{y}{x} = \frac{4}{-3} = -\frac{4}{3}$$

💡 (觀察：點 $P(-3, 4)$ 在第二像限，根據 ASTC，只有 $\sin \theta$ 為正，符合預期！)

答案： $\sin \theta = \frac{4}{5} \quad , \quad \tan \theta = -\frac{4}{3}$

6.2 三角學恒等式 (Trigonometric Identities)

無論 $\theta$ 的角度為何，以下兩個基本恒等式永遠成立，是 DSE 化簡題目的必用工具：

1. 商數關係 (Quotient Relation)

$$\tan \theta = \frac{\sin \theta}{\cos \theta}$$

2. 平方關係 (Pythagorean Identity)

$$\sin^2 \theta + \cos^2 \theta = 1$$

💡 平方關係的核心變形（極常用）：

  • $$\sin^2 \theta = 1 - \cos^2 \theta$$
  • $$\cos^2 \theta = 1 - \sin^2 \theta$$
📝 DSE 經典化簡題： 化簡 $\frac{1}{\cos^2 \theta} - \tan^2 \theta$。

步驟 1：將 $\tan \theta$ 利用商數關係展開

$$\text{原式} = \frac{1}{\cos^2 \theta} - \left(\frac{\sin \theta}{\cos \theta}\right)^2 = \frac{1}{\cos^2 \theta} - \frac{\sin^2 \theta}{\cos^2 \theta}$$

步驟 2：通分合併分子

$$= \frac{1 - \sin^2 \theta}{\cos^2 \theta}$$

步驟 3：利用平方關係變形 $1 - \sin^2 \theta = \cos^2 \theta$ 代入

$$= \frac{\cos^2 \theta}{\cos^2 \theta} = 1$$

答案： $1$

6.3 解三角方程 (Solving Trigonometric Equations)

解三角方程時，必須特別注意**像限判斷**與**根的數目**。在常見的定義域 $0^\circ \le \theta \le 360^\circ$ 內，一個方程通常會有兩個根。

⚡ 核心解題四步驟：

  1. 分離函數： 將方程整理為 $\sin \theta = k$、$\cos \theta = k$ 或 $\tan \theta = k$ 的形式。
  2. 求銳角對照角 ($\alpha$)： 忽略 $k$ 的正負號，用計算機求出基本銳角：$\alpha = \sin^{-1}|k|$（或 $\cos^{-1}|k|$ / $\tan^{-1}|k|$）。
  3. 依 ASTC 判斷像限： 根據 $k$ 原本的正負號，決定角度 $\theta$ 落在哪些像限。
  4. 計算最終角度：
    • 第一像限：$\theta = \alpha$
    • 第二像限：$\theta = 180^\circ - \alpha$
    • 第三像限：$\theta = 180^\circ + \alpha$
    • 第四像限：$\theta = 360^\circ - \alpha$
📝 經典例題： 在 $0^\circ \le \theta \le 360^\circ$ 範圍內，解方程 $2\cos \theta + 1 = 0$。

步驟 1：移項整理方程

$$2\cos \theta = -1 \implies \cos \theta = -\frac{1}{2}$$

步驟 2：求銳角對照角 $\alpha$ (取絕對值)

$$\alpha = \cos^{-1}\left|\frac{-1}{2}\right| = \cos^{-1}\left(\frac{1}{2}\right) = 60^\circ$$

步驟 3：判斷像限

因為 $\cos \theta$ 的值為 **負數**，根據 ASTC 法則，$\cos$ 在**第二像限 (Q2)** 和 **第三像限 (Q3)** 為負值。

步驟 4：代入像限公式求出所有根

第二像限根：$$\theta = 180^\circ - 60^\circ = 120^\circ$$

第三像限根：$$\theta = 180^\circ + 60^\circ = 240^\circ$$

答案： $\theta = 120^\circ \quad \text{或} \quad \theta = 240^\circ$ （共有 2 個實根）

6.4 特殊角的三角比 (Trigonometric Ratios of Special Angles)

在考試中，遇到 $30^\circ, 45^\circ, 60^\circ$ 等特殊角時，建議直接使用準確值（分數及根號）而非小數作答。以下是必須熟記的數值表：

$\theta$ $0^\circ$ $30^\circ$ $45^\circ$ $60^\circ$ $90^\circ$
$\sin \theta$ $0$ $\frac{1}{2}$ $\frac{\sqrt{2}}{2}$ $\frac{\sqrt{3}}{2}$ $1$
$\cos \theta$ $1$ $\frac{\sqrt{3}}{2}$ $\frac{\sqrt{2}}{2}$ $\frac{1}{2}$ $0$
$\tan \theta$ $0$ $\frac{\sqrt{3}}{3}$ (或 $\frac{1}{\sqrt{3}}$) $1$ $\sqrt{3}$ 未定義

💡 快速記憶小秘訣： $\sin \theta$ 從 $0^\circ$ 到 $90^\circ$ 的值，可以分別記作： $\frac{\sqrt{0}}{2}, \frac{\sqrt{1}}{2}, \frac{\sqrt{2}}{2}, \frac{\sqrt{3}}{2}, \frac{\sqrt{4}}{2}$。

6.5 相關角的三角恒等式 (Identities of Related Angles)

利用 ASTC 像限的觀念，我們可以把任何角度的三角函數轉化為第一像限的銳角 $\theta$。這在跨像限化簡時極為重要。

1. 第二像限變換 ($180^\circ - \theta$)

在第二像限 (S)，只有 $\sin$ 是正數：

  • $\sin(180^\circ - \theta) = \sin\theta$
  • $\cos(180^\circ - \theta) = -\cos\theta$
  • $\tan(180^\circ - \theta) = -\tan\theta$

2. 第三像限變換 ($180^\circ + \theta$)

在第三像限 (T)，只有 $\tan$ 是正數：

  • $\sin(180^\circ + \theta) = -\sin\theta$
  • $\cos(180^\circ + \theta) = -\cos\theta$
  • $\tan(180^\circ + \theta) = \tan\theta$

3. 第四像限變換 ($360^\circ - \theta$ 或 $-\theta$)

在第四像限 (C)，只有 $\cos$ 是正數：

  • $\sin(360^\circ - \theta) = \sin(-\theta) = -\sin\theta$
  • $\cos(360^\circ - \theta) = \cos(-\theta) = \cos\theta$
  • $\tan(360^\circ - \theta) = \tan(-\theta) = -\tan\theta$

📐 額外補充：餘角變換 ($90^\circ - \theta$)

當角度為 $90^\circ - \theta$ 時，函數名稱會互相切換 ($\sin \leftrightarrow \cos$)，且在第一像限全為正數：

  • $\sin(90^\circ - \theta) = \cos\theta$
  • $\cos(90^\circ - \theta) = \sin\theta$
  • $\tan(90^\circ - \theta) = \frac{1}{\tan\theta}$

🔄 進階變換：$y$軸相關角 ($270^\circ \pm \theta$)

與 $90^\circ$ 類似，當角度涉及 $270^\circ$ 時，函數名稱亦必須互相切換 ($\sin \leftrightarrow \cos$, $\tan \leftrightarrow \frac{1}{\tan}$)，然後根據 ASTC 決定正負號：

1. 第三像限 ($270^\circ - \theta$) - 只有 $\tan$ 為正數：

  • $\sin(270^\circ - \theta) = -\cos\theta$
  • $\cos(270^\circ - \theta) = -\sin\theta$
  • $\tan(270^\circ - \theta) = \frac{1}{\tan\theta}$

2. 第四像限 ($270^\circ + \theta$) - 只有 $\cos$ 為正數：

  • $\sin(270^\circ + \theta) = -\cos\theta$
  • $\cos(270^\circ + \theta) = \sin\theta$
  • $\tan(270^\circ + \theta) = -\frac{1}{\tan\theta}$

6.6 三角函數的圖像與特性 (Graphs of Trigonometric Functions)

要快速解題，必須在腦海中深刻印下 $\sin$、$\cos$ 和 $\tan$ 的基本圖像形狀。請特別注意它們的最大值 (Maximum)、最小值 (Minimum) 以及 週期 (Period)。

1. 正弦函數 $y = \sin \theta$

  • 極大值 (Maximum)： $1$ (發生於 $\theta=90^\circ$)
  • 極小值 (Minimum)： $-1$ (發生於 $\theta=270^\circ$)
  • 週期 (Period)： $360^\circ$ (從原點出發，呈波浪形起伏)

2. 餘弦函數 $y = \cos \theta$

  • 極大值 (Maximum)： $1$ (發生於 $\theta=0^\circ, 360^\circ$)
  • 極小值 (Minimum)： $-1$ (發生於 $\theta=180^\circ$)
  • 週期 (Period)： $360^\circ$ (從最高點出發，呈山谷形)

3. 正切函數 $y = \tan \theta$

  • 極大值 / 極小值： 無 (趨向正負無限大)
  • 週期 (Period)： $180^\circ$
  • 特徵： 圖像被漸近線 (Asymptotes) 切斷。當 $\theta = 90^\circ, 270^\circ$ 時，$\tan \theta$ 未定義。

7.1 多項式的長除法 (Long Division of Polynomials)

進行多項式長除法時，有兩個極度重要的守則：

  • 降冪排序： 被除式和除式都必須按照未知數 $x$ 的次方由大至小排列。
  • 補零項： 若果被除式缺了某個次方的項，必須補上 $0x^n$ 以保留位置，否則計算時會錯位！
📝 經典例題： 計算 $(2x^3 - 5x + 3) \div (x - 2)$ 的商式及餘式。

注意： 被除式缺少了 $x^2$ 項，我們必須將其寫成 $2x^3 \mathbf{+ 0x^2} - 5x + 3$。

長除法直式演示：

$$ \begin{array}{r|rrrr} & 2x^2 & +4x & +3 \\ \hline x - 2 & 2x^3 & +0x^2 & -5x & +3 \\ & 2x^3 & -4x^2 & & \\ \hline & & 4x^2 & -5x & \\ & & 4x^2 & -8x & \\ \hline & & & 3x & +3 \\ & & & 3x & -6 \\ \hline & & & & 9 \end{array} $$

答案： 商式 (Quotient) = $2x^2 + 4x + 3$，餘式 (Remainder) = $9$。

7.2 多項式的除法算式 (Division Algorithm)

任何多項式的除法，都可以寫成一個恆等式 (Identity)。這在解答未知數題目時非常有用！

$$\text{被除式 (Dividend)} \equiv \text{除式 (Divisor)} \times \text{商式 (Quotient)} + \text{餘式 (Remainder)}$$ $$P(x) \equiv D(x) \times Q(x) + R(x)$$

💡 重要概念： 餘式 $R(x)$ 的次數必定小於除式 $D(x)$ 的次數。
例如：若除式是二次多項式 (例如 $x^2 + 1$)，餘式最多是一次多項式，我們通常假設餘式為 $ax + b$。

7.3 餘式定理與因式定理 (Remainder Theorem & Factor Theorem)

🎯 餘式定理 (Remainder Theorem)

當一個多項式 $P(x)$ 除以一次式 $(ax - b)$ 時，其餘式 $R$ 可以直接透過代入數值求得，完全不需要做長除法！

$$\text{令除式 } ax - b = 0 \implies x = \frac{b}{a}$$ $$\text{餘式 } R = P\left(\frac{b}{a}\right)$$
📝 餘式定理應用： 求 $P(x) = 3x^3 + 2x^2 - 5x + 7$ 除以 $(x + 1)$ 的餘式。

令除式 $x + 1 = 0 \implies x = -1$

餘式 $R = P(-1) = 3(-1)^3 + 2(-1)^2 - 5(-1) + 7 = -3 + 2 + 5 + 7 = 11$。

🎯 因式定理 (Factor Theorem)

因式定理是餘式定理的特例。如果 $(ax - b)$ 是 $P(x)$ 的因式 (Factor)，這代表除得盡，即餘式為 0。

$$(ax - b) \text{ 是 } P(x) \text{ 的因式} \iff P\left(\frac{b}{a}\right) = 0$$
📝 因式定理應用： 已知 $(x - 2)$ 是 $P(x) = x^3 - kx^2 + 4x - 8$ 的因式，求 $k$ 的值。

因為 $(x - 2)$ 是因式，所以餘式 $P(2) = 0$。

$P(2) = (2)^3 - k(2)^2 + 4(2) - 8 = 0$

$8 - 4k + 8 - 8 = 0 \implies 4k = 8 \implies k = 2$

7.4 DSE 卷一 Section B 實戰：多項式長題 (參考 2022 Q14 題型)

在 DSE 卷一乙部 (Section B)，多項式通常會結合「餘式/因式定理」與「二次方程判別式」或「複數」進行跨課題考核。這是一條非常經典的長題：

🎯 終極真題模擬：
設 $p(x) = x^3 + ax^2 + bx - 6$。已知 $p(x)$ 可被 $x - 1$ 整除，且當 $p(x)$ 除以 $x + 2$ 時，餘式為 $-12$。
(a) 求 $a$ 及 $b$ 的值。 (3分)
(b) 某學生聲稱方程 $p(x) = 0$ 所有的根均為實數 (Real numbers)。你是否同意？試解釋你的答案。 (5分)

💡 題解 Part (a)：利用因式及餘式定理建立聯立方程

根據因式定理，已知 $p(1) = 0$：

$$(1)^3 + a(1)^2 + b(1) - 6 = 0 \implies a + b = 5 \quad \text{--- (第1式)}$$

根據餘式定理，已知 $p(-2) = -12$：

$$(-2)^3 + a(-2)^2 + b(-2) - 6 = -12$$

$$-8 + 4a - 2b - 6 = -12 \implies 4a - 2b = 2 \implies 2a - b = 1 \quad \text{--- (第2式)}$$

將 (第1式) + (第2式)： $(a + b) + (2a - b) = 5 + 1 \implies 3a = 6 \implies a = 2$

代 $a = 2$ 入 (第1式)： $2 + b = 5 \implies b = 3$

答案： $a = 2, \ b = 3$。由此得知 $p(x) = x^3 + 2x^2 + 3x - 6$。

💡 題解 Part (b)：降次並使用判別式 $\Delta$

要解方程 $p(x) = 0$，即 $x^3 + 2x^2 + 3x - 6 = 0$。

由 (a) 已知 $(x - 1)$ 是 $p(x)$ 的因式，我們可利用長除法將 $p(x)$ 除以 $(x - 1)$：

$$x^3 + 2x^2 + 3x - 6 = (x - 1)(x^2 + 3x + 6)$$

所以，方程可寫成：

$$(x - 1)(x^2 + 3x + 6) = 0$$

$$x - 1 = 0 \quad \text{或} \quad x^2 + 3x + 6 = 0$$

對於二次方程部分 $x^2 + 3x + 6 = 0$，檢查其判別式 $\Delta$：

$$\Delta = b^2 - 4ac = (3)^2 - 4(1)(6) = 9 - 24 = -15$$

因為 $\Delta < 0$，該二次方程沒有實根（有兩個複數根）。

結論： 原方程 $p(x) = 0$ 只有一個實根 ($x = 1$) 及兩個非實根。因此，並非所有根均為實數，不同意該學生的聲稱。

💡 基礎觀念：角與邊的命名規則

在所有解三角形的公式中，請務必緊記：「大楷代表角，細楷代表對邊」。例如，$\angle A$ 的對邊長度必然是 $a$。

8.1 三角面積公式 (Area of Triangle)

當我們已知三角形的「兩邊及夾角」(SAS, Side-Angle-Side) 時，可以利用正弦比來求面積，無需找出高：

$$\text{面積} = \frac{1}{2}ab \sin C = \frac{1}{2}bc \sin A = \frac{1}{2}ac \sin B$$
📝 經典例題： 已知 $\triangle ABC$ 中，$a = 6 \text{ cm}$，$b = 8 \text{ cm}$，且 $\angle C = 30^\circ$。求三角形面積。

代入公式：

$$\text{面積} = \frac{1}{2}(6)(8) \sin 30^\circ$$

$$\text{面積} = 24 \times 0.5 = 12 \text{ cm}^2$$

8.2 正弦公式 (Sine Formula)

適用時機： 當已知「兩角一邊」(AAS/ASA) 或「兩邊及一對角」(SSA) 時使用。

公式表示三角形中，各邊長度與其對角的正弦值成正比：

$$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C}$$

💡 解題技巧：若要求角度，可將公式倒寫為 $\frac{\sin A}{a} = \frac{\sin B}{b} = \frac{\sin C}{c}$ 會更容易列式計算！

8.3 餘弦公式 (Cosine Formula)

餘弦公式是畢氏定理的進階版，適用於任何三角形。考試中主要有兩種變形，對應不同已知條件：

1. 求邊長 (已知兩邊及夾角 SAS)

$$c^2 = a^2 + b^2 - 2ab \cos C$$

2. 求角度 (已知三邊 SSS)

將上述公式重新排列，得出求角度的專用公式：

$$\cos C = \frac{a^2 + b^2 - c^2}{2ab}$$
📝 觀念提醒：
當 $\cos C > 0$ 時，$C$ 為銳角；
當 $\cos C < 0$ 時，$C$ 為鈍角；
當 $\cos C = 0$ 時，$C = 90^\circ$（退化為直角三角形畢氏定理）。

8.4 希羅公式 (Heron's Formula)

適用時機： 當僅已知三角形的「三條邊長」(SSS)，而不需先找出任何角度時，可直接求得面積。

步驟 1： 先計算半周界 (Semi-perimeter)，記作 $s$：

$$s = \frac{a + b + c}{2}$$

步驟 2： 代入面積公式：

$$\text{面積} = \sqrt{s(s-a)(s-b)(s-c)}$$
📝 經典例題： 設三角形三邊長分別為 $a=3$, $b=4$, $c=5$。求面積。

半周界 $s = \frac{3 + 4 + 5}{2} = 6$

$$\text{面積} = \sqrt{6(6-3)(6-4)(6-5)}$$

$$\text{面積} = \sqrt{6(3)(2)(1)} = \sqrt{36} = 6$$

⚠️ 考試特大陷阱：銳角與鈍角的考慮 (Ambiguous Case)

在做題目時，只要你需要利用 正弦公式 (Sine Law) 或 三角面積公式 來反查「角度」，記緊計算機（Shift + Sin）只會顯示銳角 ($\theta$)。你必須主動考慮是否存在鈍角 ($180^\circ - \theta$) 的可能！

1. 正弦公式的「雙答案」陷阱（已知 SSA）

當已知兩邊及其中一邊的對角（例如已知 $a, b, \angle A$，求 $\angle B$）時，使用公式：

$$\sin B = \frac{b \sin A}{a}$$

經計算機求得的值為 $\theta$，此時必須列出兩個可能性：

  • 答案一（銳角）： $\angle B_1 = \theta$
  • 答案二（鈍角）： $\angle B_2 = 180^\circ - \theta$
🔍 必須進行「三角形生還者檢查」：
並非所有鈍角答案都合理，學生必須拿求得的鈍角 $\angle B_2$ 加上題目原本已知的 $\angle A$：
  • 若 $\angle B_2 + \angle A < 180^\circ$ $\rightarrow$ 鈍角答案合理！（此題有兩個可能答案，即有兩個三角形存在）。
  • 若 $\angle B_2 + \angle A \ge 180^\circ$ $\rightarrow$ 鈍角答案捨去！（因為三個內角相加已超越 $180^\circ$）。

2. 三角面積公式求角度的陷阱

當已知三角形面積以及兩條夾邊 $a, b$，要求夾角 $C$ 時，使用公式：

$$\sin C = \frac{2 \times \text{面積}}{ab}$$

同樣道理，因為 $\sin C = \sin(180^\circ - C)$，如果題目沒有註明 $\triangle ABC$ 是銳角三角形還是鈍角三角形，通常必須寫出兩個答案：

💡 示例：若經計算機算出 $\angle C = 45^\circ$：

學生必須回答：$\angle C = 45^\circ$ 或 $\angle C = 180^\circ - 45^\circ = 135^\circ$

💡 老師叮囑：只要題目叫你「求某個角度」，而你在計算過程中用過 $\sin$ 函數，就要立刻打醒十二分精神，看看題目有沒有限制角度範圍（例如註明「已知 $\angle B$ 為銳角」）。若題目毫無限制，很大機會就是要你寫齊兩個答案！

9.1 單項式及多項式的最大公因數 (H.C.F.) 及最小公倍數 (L.C.M.)

要找出多個代數式的最大公因數 (H.C.F.) 與最小公倍數 (L.C.M.)，我們會分為單項式和多項式兩種情況處理：

1. 單項式 (Monomials) 的處理方法

💡 計算規則口訣：

  • H.C.F.（搵共同，取最小）： 數字部分取最大公因數；未知數部分只取共同擁有的字母，並取其最低次方。
  • L.C.M.（全包宴，取最大）： 數字部分取最小公倍數；未知數部分收集所有出現過的字母，並取其最高次方。
📝 經典例題 1： 求 $6x^2y^3z$ 與 $9x^4y$ 的 H.C.F. 及 L.C.M.。

(a) 處理數字部分 (6 和 9)：

$\text{最大公因數}(6, 9) = 3$ ； $\text{最小公倍數}(6, 9) = 18$

(b) 處理未知數部分 ($x, y, z$)：

  • H.C.F.： $x$ 的最低次方是 $x^2$；$y$ 的最低次方是 $y^1$；$z$ 不是共同擁有，所以不取。 $\rightarrow 3x^2y$
  • L.C.M.： $x$ 的最高次方是 $x^4$；$y$ 的最高次方是 $y^3$；$z$ 有出現就要取 $z^1$。 $\rightarrow 18x^4y^3z$

✨ 最終答案： $\text{H.C.F.} = 3x^2y$ ； $\text{L.C.M.} = 18x^4y^3z$

2. 多項式 (Polynomials) 的處理方法

🛠️ 核心二部曲：

  1. 先因式分解： 必須先把所有多項式徹底因式分解（例如提取公因式、平方差公式、十字相乘法），化成一組組括號相乘的型態。
  2. 套用口訣： 把每一個「括號（因式）」視為一個整體，同樣運用「H.C.F. 取最低次方、L.C.M. 取最高次方」的原則。
📝 經典例題 2： 求 $x^2 - 9$ 與 $x^2 - 6x + 9$ 的 H.C.F. 及 L.C.M.。

步驟 1：徹底因式分解各個多項式

  • 多項式一（平方差）：$x^2 - 9 = (x - 3)(x + 3)$
  • 多項式二（十字相乘）：$x^2 - 6x + 9 = (x - 3)^2$

步驟 2：比較各個括號（因式）的次方

  • H.C.F.： 只取共同擁有的括號，並取最低次方。兩式都有 $(x - 3)$，最低是 1 次方；而 $(x + 3)$ 不是共有的，故不取。 $\rightarrow (x - 3)$
  • L.C.M.： 所有出現過的括號都要拿，並取最高次方。$(x - 3)$ 最高是 2 次方；$(x + 3)$ 最高是 1 次方。 $\rightarrow (x - 3)^2(x + 3)$

✨ 最終答案： $\text{H.C.F.} = x - 3$ ； $\text{L.C.M.} = (x - 3)^2(x + 3)$

9.2 分式化簡的技巧 (Simplification of Rational Expressions)

化簡有理分式時，絕對不可以把分子和分母加減號中間的項單獨約簡（例如：$\frac{x+2}{x}$ 絕對不能把 $x$ 約掉！）。

🛠️ 核心三大步驟：

  1. 先因式分解 (Factorization)： 先將分子和分母化成「因式相乘」的狀態（提取公因式、平方差、十字相乘）。
  2. 尋找相同因式約簡： 只有當分子和分母擁有完全相同的括號時，才可以整組約去。
  3. 通分母（適用於加減法）： 尋找分母的 L.C.M.（最小公倍數）作為共同分母。
⚠️ 考試必考：變號陷阱 (Sign Change Trick)
在因式分解中，經常會遇到 $a - b$ 與 $b - a$ 這種相反數。請謹記這個提取負號的技巧：
$$b - a = -(a - b)$$
📝 經典例題 3（分式加減與變號）： 化簡 $\frac{3}{x - 2} + \frac{2}{2 - x}$。

步驟 1：利用變號技巧統一分母

觀察到分母 $2 - x = -(x - 2)$，將負號提到分數前方：

$$\frac{3}{x - 2} - \frac{2}{x - 2}$$

步驟 2：同分母直接相加減

$$\frac{3 - 2}{x - 2} = \frac{1}{x - 2}$$

📝 進階例題 4（十字相乘與約簡）： 化簡 $\frac{x^2 - 9}{x^2 - x - 6}$。

步驟 1：分子分母各自因式分解

  • 分子（平方差）：$x^2 - 9 = (x - 3)(x + 3)$
  • 分母（十字相乘）：$x^2 - x - 6 = (x - 3)(x + 2)$

步驟 2：寫成因式型態並約簡

$$\frac{(x - 3)(x + 3)}{(x - 3)(x + 2)}$$

上下同時約去相同的因式 $(x - 3)$：

$$\text{最終答案} = \frac{x + 3}{x + 2}$$

10.1 分數次方的指數律 (Fractional Indices)

在中四階段，指數定律由整數擴展至分數（Fractional）及負數（Negative）。請緊記以下兩個核心轉換公式：

$$a^{\frac{1}{n}} = \sqrt[n]{a} \quad \text{及} \quad a^{\frac{m}{n}} = \sqrt[n]{a^m} = (\sqrt[n]{a})^m$$
$$a^{-n} = \frac{1}{a^n} \quad \text{（負指數代表移去分母）}$$
📝 經典例題 1： 不用計算機，求 $27^{-\frac{2}{3}}$ 的值。

步驟 1：先處理負指數（化成分數）

$$27^{-\frac{2}{3}} = \frac{1}{27^{\frac{2}{3}}}$$

步驟 2：將底數化為質因數的次方（$27 = 3^3$）

$$\frac{1}{27^{\frac{2}{3}}} = \frac{1}{(3^3)^{\frac{2}{3}}}$$

步驟 3：運用指數律 $(a^m)^n = a^{m \times n}$ 約簡

$$\frac{1}{3^{3 \times \frac{2}{3}}} = \frac{1}{3^2} = \frac{1}{9}$$

10.2 解指數方程的技巧 (Exponential Equations)

解指數方程主要有兩大招式。當兩邊底數不同且無法直接同步時，通常需要使用代換法（Substitution Method）化為二次方程。

🛠️ 招式一：統一底數（比較指數）

若兩邊能化成相同底數，則可直接比較指數：若 $a^x = a^y$，則 $x = y$。

🛠️ 招式二：代換法 (Substitution) ── 試卷一必考！

當看見方程同時出現 $a^{2x}$ 與 $a^x$（例如 $4^x$ 與 $2^x$），就要利用 $(a^x)^2 = a^{2x}$ 的特性進行代換。

📝 經典例題 2： 解方程 $4^x - 3(2^x) - 4 = 0$。

步驟 1：重寫方程，將 $4^x$ 化為 $(2^x)^2$

$$(2^x)^2 - 3(2^x) - 4 = 0$$

步驟 2：設 $u = 2^x$ （注意：因為指數函數必大於 0，所以 $u > 0$）

$$u^2 - 3u - 4 = 0$$

步驟 3：十字相乘法因式分解

$$(u - 4)(u + 1) = 0$$

$$u = 4 \quad \text{或} \quad u = -1$$

⚠️ 步驟 4：檢查並還原未知數

  • 當 $u = -1$ 時， $2^x = -1$ $\rightarrow$ （捨去 Rejected），因為 $2^x$ 永遠是正數。
  • 當 $u = 4$ 時， $2^x = 4 \implies 2^x = 2^2 \implies x = 2$。

✨ 最終答案： $x = 2$

10.3 指數函數的圖像特性 ($y = a^x$)

對於指數函數 $y = a^x$（其中 $a > 0$ 且 $a \neq 1$），底數 $a$ 的範圍直接決定了圖像的走向：

🔍 兩者必考共同特徵（MC常問）：

  1. 必過固定點： 無論 $a$ 是多少，當 $x = 0$ 時 $y = a^0 = 1$，因此圖像的 $y$ 截距必然是 $1$（恆過 $(0,1)$）。
  2. 圖像位置： 圖像永遠位於 $x$ 軸的上方（即 $y > 0$），絕不與 $x$ 軸相交。
  3. 水平漸近線 (Horizontal Asymptote)： $x$ 軸（即方程 $y = 0$）是圖像的水平漸近線。當圖像無限延伸時，只會無限逼近 $x$ 軸但絕不觸碰。

11.1 鞏固代換法的技巧 (Substitution Method)

有些方程表面上不是二次方程（例如有四次方、分數、或者根號），但我們只要把它們「偽裝」的部分拆解，利用代換法（設 $u = \dots$），就可以將其降階為我們熟悉的二次方程來求解。

常見的代換類型：

  • 雙二次方程： 見到 $x^4$ 與 $x^2$，設 $u = x^2$
  • 含有根號： 見到 $x$ 與 $\sqrt{x}$，設 $u = \sqrt{x}$
  • 重複出現的代數式： 見到同一組長長的括號重複，直接將整個括號設為 $u$
📝 經典例題 1（雙二次方程）： 解方程 $x^4 - 5x^2 + 4 = 0$。

步驟 1：運用代換法降階

設 $u = x^2$。方程變為：

$$u^2 - 5u + 4 = 0$$

步驟 2：解這個新的二次方程

$$(u - 1)(u - 4) = 0$$

$$u = 1 \quad \text{或} \quad u = 4$$

步驟 3：還原未知數 $x$（⚠️ 學生最常忘記的步驟！）

將 $u$ 變回 $x^2$：

當 $x^2 = 1$ 時， $x = \pm 1$

當 $x^2 = 4$ 時， $x = \pm 2$

✨ 最終答案： $x = 1, -1, 2, -2$ （共有四個解）

11.2 任意直線與二次方程的解及解的數量

當我們想找出「一條直線」與「一條拋物線（二次函數圖像）」是否有相交、以及交點在哪裡時，我們可以將兩者的方程聯立（Simultaneous Equations）。

💡 解題概念：

設直線方程為 $y = mx + c_1$，二次方程為 $y = ax^2 + bx + c_2$。
將兩者相等： $ax^2 + bx + c_2 = mx + c_1$
把它整理成標準二次方程格式： $ax^2 + (b-m)x + (c_2-c_1) = 0$

交點數量與判別式 ($\Delta$) 的關係：

整理出新的二次方程後，我們可以利用它的判別式 $\Delta = B^2 - 4AC$ 來判斷這條直線和拋物線的幾何關係：

📝 經典例題 2（求聯立方程的交點 ── 兩點相交）：
求直線 $y = 2x + 2$ 與拋物線 $y = x^2 - x - 2$ 的交點坐標。

步驟 1：聯立方程（將線性方程代入二次方程中）

由於兩個方程都等於 $y$，我們可以直接建立等式：

$$x^2 - x - 2 = 2x + 2$$

步驟 2：移項整理，化為標準一般式 ($ax^2 + bx + c = 0$)

將右邊的項移到左邊：

$$x^2 - 3x - 4 = 0$$

步驟 3：因式分解並求解 $x$

利用十字相乘法：

$$(x - 4)(x + 1) = 0$$

$$x = 4 \quad \text{或} \quad x = -1$$

步驟 4：將 $x$ 的值代回直線方程，求出對應的 $y$ 坐標

  • 當 $x = 4$ 時： $y = 2(4) + 2 = 10 \quad \rightarrow \text{第一個交點為 } (4, 10)$
  • 當 $x = -1$ 時： $y = 2(-1) + 2 = 0 \quad \rightarrow \text{第二個交點為 } (-1, 0)$

✨ 最終答案： 直線與拋物線的交點坐標為 $(4, 10)$ 及 $(-1, 0)$。

📝 經典例題 3（利用 $\Delta$ 證明交點數量 ── 相切）：
已知拋物線 $y = x^2 - 4x + 5$ 及 直線 $y = 2x - 4$。證明它們只相交於一點，並求出該交點坐標。

步驟 1：聯立方程

$$x^2 - 4x + 5 = 2x - 4$$

步驟 2：整理成一般式

$$x^2 - 6x + 9 = 0$$

步驟 3：利用判別式 $\Delta$ 證明交點數量

這裡 $A = 1, B = -6, C = 9$：

$$\Delta = B^2 - 4AC = (-6)^2 - 4(1)(9) = 36 - 36 = 0$$

因為 $\Delta = 0$，證明了直線與拋物線相切（即兩者只相交於一點）。

步驟 4：解方程找出唯一的交點坐標

$$(x - 3)^2 = 0 \implies x = 3$$

將 $x = 3$ 代入直線方程： $y = 2(3) - 4 = 2$

✨ 最終答案： 唯一的交點（切點）坐標為 $(3, 2)$。
